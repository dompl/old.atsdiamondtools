== Description ==
Woocommerce Plugin that allows you to select how "Free Shipping" coupons will be applied on checkout.

How to use:
On settings page "WooCommerce > Shipping > Shipping options", select the behavor.
Add coupon as usually, check box "Allow free shipping", Save and use coupon as usually.


== Features ==
* Can Add discount with amount that is equal to the chosen shipping price.
* Can Null the prices of available shipping methods.
* Can Hide all shipping methods, except "Free Shipping"
* Can Hide all shipping methods, except "Free Shipping" and "Local Pickup"
* WPML compatible.
* Polylang compatible.


== Installation ==

1. Upload plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use [job-postings] to print jobs listing
